## 使用方法：
 #### 1、引入依赖包（本地依赖库路径：raindow/kuzwlu/）
    <dependency>
            <groupId>rainbow.kuzwlu</groupId>
            <artifactId>dynamic-datasource-boot-starter</artifactId>
            <version>1.2</version>
    </dependency>
